# TiDBクラウドロードマップ

TiDB Cloudのロードマップでは、近い将来に来るものを提供しています。これにより、新機能や改善点を事前に確認し、進捗状況を追い、途中の主要なマイルストーンについて学ぶことができます。開発の過程で、このロードマップはユーザーのニーズ、フィードバック、および当社の評価に基づいて変更される可能性があります。

✅: 機能または改善がTiDB Cloudで利用可能です。
